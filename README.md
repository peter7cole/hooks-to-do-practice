# Coding Garden To Do App with Hooks

**Author** Peter Cole  
**Version** 1.0.3

## Description

- Code along with [Coding Garden](https://coding.garden/#/)'s [video](https://www.youtube.com/watch?v=9eSwZ0z-50c)
